  <implementation>
Create `src/engine/calculators.ts` with:

1. Import types, constants, formulas
2. Helper function `computeTotals(sizing: ClusterSizing): { vcpu, ramGB, storageGB }` that sums across all non-null NodeSpec entries (count *vcpu, count* ramGB, count * storageGB)
3. `calcStandardHA(config: ClusterConfig): { sizing: ClusterSizing; warnings: ValidationWarning[] }`:
   - Compute workers via `workerCount()` using config.workload fields
   - Compute CP via `cpSizing(workers)`
   - masterNodes = { count: 3, ...cpSpec, storageGB: 100 }
   - workerNodes = { count: workers, vcpu: config.workload.nodeVcpu, ramGB: config.workload.nodeRamGB, storageGB: 100 }
   - infraNodes: if config.addOns.infraNodesEnabled, use infraNodeSizing(workers) with count=3, else null
   - odfNodes: null (handled by addons.ts separately)
   - rhacmWorkers: null (handled by addons.ts separately)
   - Compute totals

4. `calcCompact3Node(config)`: 3 combined CP nodes, workerNodes=null, add subscription warning
5. `calcSNO(config)`: lookup SNO_STD_MIN/SNO_EDGE_MIN/SNO_TELECOM_MIN by config.snoProfile, workerNodes=null
6. `calcTNA(config)`: 2 CP nodes as masterNodes, arbiter as workerNodes={count:1, vcpu:2, ramGB:8, storageGB:50}, add TP warning
7. `calcTNF(config)`: 2 CP nodes as masterNodes, workerNodes=null, add TP warning + Redfish warning
8. `calcHCP(config)`: Calculate per-hosted-cluster resource needs using HCP constants + QPS scaling. Management cluster workers = ceil(totalCpu / (allocatable * 0.70)). masterNodes = management CP (3 nodes). workerNodes = management workers.
9. `calcMicroShift(config)`: 1 node = MICROSHIFT_SYS_MIN + workload overhead (ceil(totalPodCpu/1000) vCPU, ceil(totalPodMem/1024) GB RAM added to base 2/2)
10. `calcManagedCloud(config)`: all count=0, warning 'validation.managedCloudNoHardware'
11. `calcCluster(config)`: exhaustive switch dispatcher

Each calc function returns `{ sizing: ClusterSizing; warnings: ValidationWarning[] }`. The `calcCluster` dispatcher calls the topology function and assembles the final result.

Validation warning message keys (i18n, no spaces):

- `validation.compactSubscriptionNote`
- `validation.techPreviewNotForProduction`
- `validation.redfishBmcRequired`
- `validation.managedCloudNoHardware`
